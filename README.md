## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Start dev server at http://localhost:5173
npm run dev
```

### Production Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```
